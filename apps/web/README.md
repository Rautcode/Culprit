# apps/web — Culprit UI

Build step 8 (SPEC_VERSION.md): Incident List + Incident Detail, per
[docs/08-ui-design.md](../../docs/08-ui-design.md). Next.js App Router,
TypeScript, Tailwind. Dark theme.

## Data source

`lib/incidents.json` is **real pipeline output**, generated by
[`scripts/export_incidents.py`](../../scripts/export_incidents.py) — every
harness scenario run through the actual correlation pipeline with
leave-one-out incident memory. Nothing rendered here is mocked UI data.
The JSON matches the response shapes the Phase 2 REST API will serve
(`lib/types.ts` documents them); swapping to the live API is a change in
`lib/data.ts`, not in the screens.

Regenerate after pipeline/scenario changes:

```
python scripts/export_incidents.py
```

## Run

```
npm install
npm run dev     # http://localhost:3000
npm run build   # static build — all incident pages prerendered
```
