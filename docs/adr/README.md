# Architecture Decision Records

Any change to a frozen item in [SPEC_VERSION.md](../../SPEC_VERSION.md) —
architecture, rule engine, knowledge graph, confidence formula, evaluation
metrics, or success criteria — gets an ADR here (`NNNN-short-title.md`)
explaining the why, and a corresponding row in SPEC_VERSION.md's amendment
log. No silent edits to the frozen spec.

Template: Context → Decision → Consequences → Alternatives considered.
